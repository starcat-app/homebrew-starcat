# Security Policy

## Reporting a vulnerability

Report Cask tampering, checksum mismatches, compromised download URLs, or release-signing concerns through [GitHub Security Advisories](https://github.com/starcat-app/homebrew-starcat/security/advisories/new). Do not open a public issue containing credentials, exploit details, or unpublished release URLs.

## Supported versions

The current Cask on the default branch is supported. The Cask must reference HTTPS assets published by Starcat and an exact SHA-256 checksum for the notarized release artifact.
