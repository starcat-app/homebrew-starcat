# Homebrew Starcat

Homebrew tap for installing Starcat on macOS.

Starcat is a native macOS app that turns GitHub Stars into a searchable AI knowledge base.

## Install

```bash
brew tap dong4j/starcat
brew install --cask starcat
```

## Upgrade

```bash
brew update
brew upgrade --cask starcat
```

## Uninstall

```bash
brew uninstall --cask starcat
```

## Requirements

- macOS 15 Sequoia or newer.
- Apple Silicon Mac.
